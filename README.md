## Highlights

- 🎨 **Custom Icons** — Change any app icon via GUI or CLI, with [macOSicons.com](https://macosicons.com/) integration
- 🖥️ **Dock Preview** — Live preview of your Dock with custom icons
- 📂 **Folder Icons** — Customize folder icons too
- 🔄 **Background Service** — Auto-restore icons on schedule or after app updates
- ⌨️ **CLI Tool** — Full-featured command-line interface for scripting and automation
- ⭐ **Favorites & History** — Save and revisit your icon choices
- 🌍 **30 Languages** — Localized worldwide

## Quick Install

### Homebrew

```bash
brew install Bengerthelorf/tap/iconchanger
```

### Manual

Download the latest DMG from [Releases](https://github.com/Bengerthelorf/macIconChanger/releases/latest), drag to Applications, and launch.

> **Need help?** Check the [Getting Started](https://bengerthelorf.github.io/macIconChanger/guide/getting-started) guide, or browse the full [Documentation](https://bengerthelorf.github.io/macIconChanger/).

## System Requirements

- macOS 13.0+
- Administrator privileges
