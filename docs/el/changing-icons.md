## Χρήση της γραφικής διεπαφής

### Αναζήτηση στο διαδίκτυο

1. Επιλέξτε μια εφαρμογή από την πλαϊνή μπάρα.
2. Περιηγηθείτε στα εικονίδια από το [macOSicons.com](https://macosicons.com/) στην κύρια περιοχή.
3. Χρησιμοποιήστε το αναπτυσσόμενο μενού **Style** για φιλτράρισμα κατά στυλ (π.χ. Liquid Glass).
4. Κάντε κλικ σε ένα εικονίδιο για να το εφαρμόσετε.

![Αναζήτηση εικονιδίων](/images/search-icons.png)

### Επιλογή τοπικού αρχείου

Κάντε κλικ στο **Choose from the Local** (ή πατήστε <kbd>Cmd</kbd>+<kbd>O</kbd>) για να ανοίξετε τον επιλογέα αρχείων. Υποστηριζόμενες μορφές: PNG, JPEG, ICNS, TIFF, HEIC, WebP, BMP, GIF, SVG.

### Drag & Drop

Σύρετε ένα αρχείο εικόνας από το Finder απευθείας στην περιοχή εικονιδίου της εφαρμογής. Θα εμφανιστεί μπλε επισήμανση για επιβεβαίωση της ζώνης αφέσεως.

![Drag and drop](/images/drag-drop.png)

### Επαναφορά προεπιλεγμένου εικονιδίου

Για να επαναφέρετε το αρχικό εικονίδιο μιας εφαρμογής:
- Κάντε κλικ στο κουμπί **Restore Default** (ή πατήστε <kbd>Cmd</kbd>+<kbd>Delete</kbd>)
- Ή κάντε δεξί κλικ στην εφαρμογή στην πλαϊνή μπάρα και επιλέξτε **Restore Default Icon**

## Απόδραση από τη φυλακή squircle (macOS Tahoe)

Το macOS 26 Tahoe αναγκάζει όλα τα εικονίδια εφαρμογών να λάβουν σχήμα squircle (στρογγυλεμένο τετράγωνο). Εφαρμογές με μη συμβατά εικονίδια συρρικνώνονται και τοποθετούνται σε γκρι φόντο squircle.

Το IconChanger μπορεί να το διορθώσει αυτό εφαρμόζοντας εκ νέου το ενσωματωμένο εικονίδιο μιας εφαρμογής ως προσαρμοσμένο εικονίδιο, παρακάμπτοντας έτσι την επιβολή squircle του macOS.

### Ανά εφαρμογή

Κάντε δεξί κλικ σε μια εφαρμογή στην πλαϊνή μπάρα και επιλέξτε **Escape Squircle Jail**.

### Όλες οι εφαρμογές ταυτόχρονα

Κάντε κλικ στο μενού **⋯** στη γραμμή εργαλείων και επιλέξτε **Escape Squircle Jail (All Apps)**. Αυτό επεξεργάζεται όλες τις εφαρμογές που δεν έχουν ήδη προσαρμοσμένα εικονίδια.

::: tip
Τα προσαρμοσμένα εικονίδια που ορίζονται με αυτόν τον τρόπο **δεν** υποστηρίζουν τις λειτουργίες Clear, Tinted ή Dark εικονιδίων του macOS Tahoe — παραμένουν στατικά. Αυτός είναι περιορισμός του συστήματος.
:::

::: info
Η υπηρεσία παρασκηνίου σας θα εφαρμόσει αυτόματα ξανά τα εικονίδια μετά από ενημερώσεις εφαρμογών, κρατώντας τα εκτός της φυλακής squircle.
:::

## Εικονίδια φακέλων

Μπορείτε επίσης να προσαρμόσετε τα εικονίδια φακέλων. Προσθέστε φακέλους μέσω **Settings** > **Application** > **Application Folders**, ή κάντε κλικ στο κουμπί **+** στην ενότητα φακέλων της πλαϊνής μπάρας.

Αφού προστεθεί ένας φάκελος, εμφανίζεται στην πλαϊνή μπάρα όπως οι εφαρμογές. Μπορείτε να αναζητήσετε εικονίδια, να σύρετε και αφήσετε εικόνες ή να επιλέξετε τοπικά αρχεία — η ίδια ροή εργασίας με την αλλαγή εικονιδίων εφαρμογών.

::: tip
Ονόματα φακέλων όπως "go" ή "Downloads" μπορεί να μην δώσουν καλά αποτελέσματα αναζήτησης στο macOSicons.com. Χρησιμοποιήστε [ψευδώνυμα](./aliases) για να ορίσετε ένα πιο κατάλληλο όνομα αναζήτησης (π.χ. ορίστε ψευδώνυμο "Documents" σε "folder").
:::

## Προσωρινή αποθήκευση εικονιδίων

Όταν εφαρμόζετε ένα προσαρμοσμένο εικονίδιο, αποθηκεύεται αυτόματα στην κρυφή μνήμη. Αυτό σημαίνει:
- Τα προσαρμοσμένα εικονίδιά σας μπορούν να επαναφερθούν μετά από ενημερώσεις εφαρμογών
- Η υπηρεσία παρασκηνίου μπορεί να τα εφαρμόσει ξανά σε προγραμματισμένα διαστήματα
- Μπορείτε να εξάγετε και να εισάγετε τις ρυθμίσεις εικονιδίων σας

Διαχειριστείτε τα αποθηκευμένα εικονίδια στις **Settings** > **Icon Cache**.

## Συντομεύσεις πληκτρολογίου

| Συντόμευση | Ενέργεια |
|---|---|
| <kbd>Cmd</kbd>+<kbd>O</kbd> | Επιλογή τοπικού αρχείου εικονιδίου |
| <kbd>Cmd</kbd>+<kbd>Delete</kbd> | Επαναφορά προεπιλεγμένου εικονιδίου |
| <kbd>Cmd</kbd>+<kbd>R</kbd> | Ανανέωση εμφάνισης εικονιδίου |

## Συμβουλές

- Αν δεν βρεθούν εικονίδια για μια εφαρμογή, δοκιμάστε να [ορίσετε ένα ψευδώνυμο](./aliases) με πιο απλό όνομα.
- Ο μετρητής (π.χ. "12/15") δείχνει πόσα εικονίδια φορτώθηκαν επιτυχώς από τα συνολικά που βρέθηκαν.
- Τα εικονίδια ταξινομούνται κατά δημοτικότητα (αριθμός λήψεων) από προεπιλογή.
