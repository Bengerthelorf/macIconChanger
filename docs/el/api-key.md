Ένα κλειδί API από το [macosicons.com](https://macosicons.com/) απαιτείται για αναζήτηση εικονιδίων στο διαδίκτυο. Χωρίς αυτό, μπορείτε να χρησιμοποιείτε τοπικά αρχεία εικόνων.

## Απόκτηση κλειδιού API

1. Επισκεφτείτε το [macosicons.com](https://macosicons.com/) και δημιουργήστε λογαριασμό.
2. Ζητήστε ένα κλειδί API από τις ρυθμίσεις του λογαριασμού σας.
3. Αντιγράψτε το κλειδί.

![Πώς να αποκτήσετε κλειδί API](/images/api-key.png)

## Εισαγωγή του κλειδιού

1. Ανοίξτε το IconChanger.
2. Μεταβείτε στις **Settings** > **Advanced**.
3. Επικολλήστε το κλειδί API σας στο πεδίο **API Key**.
4. Κάντε κλικ στο **Test Connection** για να επαληθεύσετε ότι λειτουργεί.

![Ρυθμίσεις κλειδιού API](/images/api-key-settings.png)

## Χρήση χωρίς κλειδί API

Μπορείτε να αλλάξετε εικονίδια εφαρμογών χωρίς κλειδί API με τους εξής τρόπους:

- Χρήση τοπικών αρχείων εικόνων (κάντε κλικ στο **Choose from the Local** ή σύρετε και αφήστε μια εικόνα)
- Χρήση εικονιδίων που περιλαμβάνονται στην ίδια την εφαρμογή (εμφανίζονται στην ενότητα "Local")

## Προηγμένες ρυθμίσεις API

Στις **Settings** > **Advanced** > **API Settings**, μπορείτε να ρυθμίσετε με ακρίβεια τη συμπεριφορά του API:

| Ρύθμιση | Προεπιλογή | Περιγραφή |
|---|---|---|
| **Retry Count** | 0 (χωρίς επανάληψη) | Πόσες φορές να επαναληφθεί ένα αποτυχημένο αίτημα (0–3) |
| **Timeout** | 15 δευτερόλεπτα | Χρονικό όριο ανά προσπάθεια αιτήματος |
| **Monthly Limit** | 50 | Μέγιστος αριθμός ερωτημάτων API ανά μήνα |

Ο μετρητής **Monthly Usage** εμφανίζει την τρέχουσα χρήση σας. Μηδενίζεται αυτόματα την 1η κάθε μήνα ή μπορείτε να τον μηδενίσετε χειροκίνητα.

### Προσωρινή μνήμη αναζήτησης εικονιδίων

Ενεργοποιήστε το **Cache API Results** για αποθήκευση αποτελεσμάτων αναζήτησης στον δίσκο. Τα αποθηκευμένα αποτελέσματα διατηρούνται μετά την επανεκκίνηση της εφαρμογής, μειώνοντας τη χρήση του API. Χρησιμοποιήστε το κουμπί ανανέωσης κατά την περιήγηση εικονιδίων για ανάκτηση ενημερωμένων αποτελεσμάτων.

## Αντιμετώπιση προβλημάτων

Αν ο έλεγχος API αποτύχει:
- Βεβαιωθείτε ότι το κλειδί σας είναι σωστό (χωρίς επιπλέον κενά)
- Ελέγξτε τη σύνδεσή σας στο διαδίκτυο
- Το API του macosicons.com μπορεί να είναι προσωρινά μη διαθέσιμο
