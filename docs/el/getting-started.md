## Απαιτήσεις

- macOS 13.0 (Ventura) ή νεότερο
- Δικαιώματα διαχειριστή (για την αρχική ρύθμιση και την αλλαγή εικονιδίων)

## Εγκατάσταση

### Homebrew (Συνιστάται)

```bash
brew install Bengerthelorf/tap/iconchanger
```

### Χειροκίνητη λήψη

1. Κατεβάστε το τελευταίο DMG από τις [εκδόσεις στο GitHub](https://github.com/Bengerthelorf/macIconChanger/releases/latest).
2. Ανοίξτε το DMG και σύρετε το **IconChanger** στον φάκελο Εφαρμογές.
3. Εκκινήστε το IconChanger.

## Πρώτη εκκίνηση

Κατά την πρώτη εκκίνηση, το IconChanger θα σας ζητήσει να ολοκληρώσετε μια εφάπαξ ρύθμιση δικαιωμάτων. Αυτό απαιτείται για να μπορεί η εφαρμογή να αλλάζει εικονίδια εφαρμογών.

![Οθόνη αρχικής ρύθμισης](/images/setup-prompt.png)

Κάντε κλικ στο κουμπί ρύθμισης και εισαγάγετε τον κωδικό διαχειριστή σας. Το IconChanger θα ρυθμίσει αυτόματα τα απαραίτητα δικαιώματα (έναν κανόνα sudoers για το βοηθητικό σενάριο).

::: tip
Αν η αυτόματη ρύθμιση αποτύχει, δείτε τις [οδηγίες χειροκίνητης ρύθμισης](./setup).
:::

## Αλλαγή του πρώτου σας εικονιδίου

1. Επιλέξτε μια εφαρμογή από την πλαϊνή μπάρα.
2. Περιηγηθείτε στα εικονίδια από το [macOSicons.com](https://macosicons.com/) ή επιλέξτε ένα τοπικό αρχείο εικόνας.
3. Κάντε κλικ σε ένα εικονίδιο για να το εφαρμόσετε.

![Κύρια διεπαφή](/images/main-interface.png)

Αυτό ήταν! Το εικονίδιο της εφαρμογής θα αλλάξει αμέσως.
