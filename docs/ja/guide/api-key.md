# API キー

オンラインでアイコンを検索するには、[macosicons.com](https://macosicons.com/) の API キーが必要です。API キーがなくても、ローカルの画像ファイルは引き続き使用できます。

## API キーの取得

1. [macosicons.com](https://macosicons.com/) にアクセスしてアカウントを作成します。
2. アカウント設定から API キーをリクエストします。
3. キーをコピーします。

![API キーの取得方法](/images/api-key.png)

## キーの入力

1. IconChanger を開きます。
2. **Settings** > **Advanced** に移動します。
3. **API Key** フィールドに API キーを貼り付けます。
4. **Test Connection** をクリックして動作を確認します。

![API キー設定](/images/api-key-settings.png)

## API キーなしで使用する

API キーがなくても、以下の方法でアプリアイコンを変更できます：

- ローカルの画像ファイルを使用する（**Choose from the Local** をクリックするか、画像をドラッグ＆ドロップ）
- アプリ内にバンドルされたアイコンを使用する（「Local」セクションに表示されます）

## トラブルシューティング

API テストが失敗する場合：
- キーが正しいか確認してください（余分なスペースがないか）
- インターネット接続を確認してください
- macosicons.com の API が一時的に利用できない可能性があります
