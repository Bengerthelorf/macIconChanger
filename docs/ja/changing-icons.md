## GUI を使用する

### オンライン検索

1. サイドバーからアプリを選択します。
2. メインエリアで [macOSicons.com](https://macosicons.com/) のアイコンを閲覧します。
3. **Style** ドロップダウンでスタイル（例：Liquid Glass）をフィルタリングします。
4. アイコンをクリックして適用します。

![アイコンの検索](/images/search-icons.png)

### ローカルファイルを選択

**Choose from the Local**（または <kbd>Cmd</kbd>+<kbd>O</kbd>）をクリックしてファイルピッカーを開きます。対応形式：PNG、JPEG、ICNS、TIFF、HEIC、WebP、BMP、GIF、SVG。

### ドラッグ＆ドロップ

Finder から画像ファイルをアプリのアイコンエリアに直接ドラッグします。ドロップゾーンを示す青いハイライトが表示されます。

![ドラッグ＆ドロップ](/images/drag-drop.png)

### デフォルトアイコンに戻す

アプリの元のアイコンに戻すには：
- **Restore Default** ボタンをクリック（または <kbd>Cmd</kbd>+<kbd>Delete</kbd> を押す）
- または、サイドバーでアプリを右クリックして **Restore Default Icon** を選択

## Squircle Jail からの脱出（macOS Tahoe）

macOS 26 Tahoe では、すべてのアプリアイコンが squircle（角丸四角形）の形状に強制されます。非準拠のアイコンを持つアプリは縮小され、グレーの squircle 背景の上に配置されます。

IconChanger は、アプリ自体にバンドルされたアイコンをカスタムアイコンとして再適用することで、macOS の squircle 強制をバイパスし、この問題を解決できます。

### アプリごとに適用

サイドバーでアプリを右クリックし、**Escape Squircle Jail** を選択します。

### すべてのアプリに一括適用

ツールバーの **⋯** メニューをクリックし、**Escape Squircle Jail (All Apps)** を選択します。カスタムアイコンが設定されていないすべてのアプリが処理されます。

::: tip
この方法で設定されたカスタムアイコンは、macOS Tahoe の Clear、Tinted、Dark アイコンモードに対応して**いません**。アイコンは静的なままです。これはシステムの制限です。
:::

::: info
バックグラウンドサービスはアプリの更新後にアイコンを自動的に再適用し、squircle jail からの脱出状態を維持します。
:::

## フォルダアイコン

フォルダのアイコンもカスタマイズできます。**Settings** > **Application** > **Application Folders** からフォルダを追加するか、サイドバーのフォルダセクションにある **+** ボタンをクリックしてください。

フォルダを追加すると、アプリと同様にサイドバーに表示されます。アイコンの検索、画像のドラッグ＆ドロップ、ローカルファイルの選択など、アプリのアイコンを変更する場合と同じワークフローで操作できます。

::: tip
「go」や「Downloads」のようなフォルダ名は、macOSicons.com で良い検索結果が得られない場合があります。[エイリアス](./aliases)を使って、より検索しやすい名前を設定してください（例：「Documents」のエイリアスを「folder」に設定）。
:::

## アイコンのキャッシュ

カスタムアイコンを適用すると、自動的にキャッシュされます。これにより：
- アプリの更新後にカスタムアイコンを復元できます
- バックグラウンドサービスがスケジュールに従って再適用できます
- アイコン設定をエクスポート・インポートできます

キャッシュされたアイコンは **Settings** > **Icon Cache** で管理できます。

## キーボードショートカット

| ショートカット | アクション |
|---|---|
| <kbd>Cmd</kbd>+<kbd>O</kbd> | ローカルのアイコンファイルを選択 |
| <kbd>Cmd</kbd>+<kbd>Delete</kbd> | デフォルトアイコンに戻す |
| <kbd>Cmd</kbd>+<kbd>R</kbd> | アイコン表示を更新 |

## ヒント

- アプリのアイコンが見つからない場合は、よりシンプルな名前で[エイリアスを設定](./aliases)してみてください。
- カウンター（例：「12/15」）は、見つかった総数のうち正常に読み込まれたアイコン数を示します。
- アイコンはデフォルトで人気順（ダウンロード数）でソートされます。
