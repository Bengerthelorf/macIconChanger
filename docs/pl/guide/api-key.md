# Klucz API

Klucz API z [macosicons.com](https://macosicons.com/) jest wymagany do wyszukiwania ikon online. Bez niego nadal możesz używać lokalnych plików obrazów.

## Uzyskiwanie klucza API

1. Odwiedź [macosicons.com](https://macosicons.com/) i załóż konto.
2. Poproś o klucz API w ustawieniach konta.
3. Skopiuj klucz.

![Jak uzyskać klucz API](/images/api-key.png)

## Wprowadzanie klucza

1. Otwórz IconChanger.
2. Przejdź do **Settings** > **Advanced**.
3. Wklej klucz API w polu **API Key**.
4. Kliknij **Test Connection**, aby sprawdzić, czy działa.

![Ustawienia klucza API](/images/api-key-settings.png)

## Korzystanie bez klucza API

Możesz zmieniać ikony aplikacji bez klucza API, korzystając z:

- Lokalnych plików obrazów (kliknij **Choose from the Local** lub przeciągnij i upuść obraz)
- Ikon dołączonych do samej aplikacji (wyświetlanych w sekcji "Local")

## Rozwiązywanie problemów

Jeśli test API się nie powiedzie:
- Sprawdź, czy klucz jest poprawny (bez dodatkowych spacji)
- Zweryfikuj połączenie internetowe
- API macosicons.com może być tymczasowo niedostępne
