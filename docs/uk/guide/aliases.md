# Псевдоніми програм

Деякі програми мають назви, які погано відповідають пошуковим запитам для іконок. Наприклад, "Adobe Photoshop 2024" може не дати гарних результатів, але "Photoshop" — дасть. Псевдоніми вирішують цю проблему, пов'язуючи альтернативну пошукову назву з програмою.

## Встановлення псевдоніма

1. Клацніть правою кнопкою миші на програмі в бічній панелі.
2. Оберіть **Set the Alias**.
3. Введіть зручну для пошуку назву (наприклад, `Photoshop` замість `Adobe Photoshop 2024`).

![Встановлення псевдоніма](/images/set-alias.png)

Тепер програма буде використовувати назву псевдоніма при пошуку іконок на macOSicons.com.

## Керування псевдонімами

Переглядайте та редагуйте всі псевдоніми в **Settings** > **Application** > **Alias Names**.

![Налаштування псевдонімів](/images/alias-settings.png)

## Вбудовані псевдоніми

IconChanger містить деякі вбудовані псевдоніми для поширених програм:
- `PyCharm Professional Edition` → `PyCharm`
- `Discord PTB` → `Discord`

## Поради

- Використовуйте короткі та загальні назви псевдонімів для кращих результатів пошуку.
- Псевдоніми включаються при [експорті конфігурації](./import-export).
