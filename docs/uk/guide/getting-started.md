# Швидкий старт

## Вимоги

- macOS 13.0 (Ventura) або новіша версія
- Права адміністратора (для початкового налаштування та зміни іконок)

## Встановлення

### Homebrew (рекомендовано)

```bash
brew install Bengerthelorf/tap/iconchanger
```

### Завантаження вручну

1. Завантажте останній DMG з [GitHub Releases](https://github.com/Bengerthelorf/macIconChanger/releases/latest).
2. Відкрийте DMG та перетягніть **IconChanger** до теки «Програми».
3. Запустіть IconChanger.

## Перший запуск

При першому запуску IconChanger запропонує виконати одноразове налаштування дозволів. Це необхідно для того, щоб програма могла змінювати іконки.

![Екран налаштування при першому запуску](/images/setup-prompt.png)

Натисніть кнопку налаштування та введіть пароль адміністратора. IconChanger автоматично сконфігурує необхідні дозволи (правило sudoers для допоміжного скрипта).

::: tip
Якщо автоматичне налаштування не вдалося, див. [Початкове налаштування](./setup) для інструкцій з ручного налаштування.
:::

## Зміна першої іконки

1. Оберіть програму з бічної панелі.
2. Перегляньте іконки з [macOSicons.com](https://macosicons.com/) або виберіть локальний файл зображення.
3. Натисніть на іконку, щоб застосувати її.

![Головний інтерфейс](/images/main-interface.png)

Ось і все! Іконку програми буде змінено миттєво.
