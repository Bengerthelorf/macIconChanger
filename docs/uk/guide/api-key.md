# API-ключ

API-ключ від [macosicons.com](https://macosicons.com/) потрібен для пошуку іконок онлайн. Без нього ви все одно можете використовувати локальні файли зображень.

## Отримання API-ключа

1. Відвідайте [macosicons.com](https://macosicons.com/) та створіть обліковий запис.
2. Запросіть API-ключ у налаштуваннях облікового запису.
3. Скопіюйте ключ.

![Як отримати API-ключ](/images/api-key.png)

## Введення ключа

1. Відкрийте IconChanger.
2. Перейдіть до **Settings** > **Advanced**.
3. Вставте ваш API-ключ у поле **API Key**.
4. Натисніть **Test Connection**, щоб перевірити його працездатність.

![Налаштування API-ключа](/images/api-key-settings.png)

## Використання без API-ключа

Ви все одно можете змінювати іконки програм без API-ключа, використовуючи:

- Локальні файли зображень (натисніть **Choose from the Local** або перетягніть зображення)
- Іконки, вбудовані в саму програму (показані в розділі "Local")

## Усунення проблем

Якщо перевірка API не вдалася:
- Переконайтеся, що ваш ключ правильний (без зайвих пробілів)
- Перевірте з'єднання з інтернетом
- API macosicons.com може бути тимчасово недоступним
