# 顯示設定

在 **設定** > **顯示** 中配置 IconChanger 的視覺外觀。

## 外觀

在 **系統**、**淺色** 或 **深色** 模式之間選擇。系統選項會跟隨你的 macOS 外觀設定。

### 自訂圖示徽章

開啟 **顯示自訂圖示徽章** 後，側邊欄中已套用自訂圖示的應用程式會顯示一個綠色勾號標記。

## Dock 預覽

Dock 預覽會即時展示自訂圖示在 macOS Dock 中的效果。

### 預覽模式

- **僅在 Dock 篩選時** — 僅在按 Dock 應用程式篩選時顯示 Dock 預覽
- **始終** — 始終在詳細資訊區域頂部顯示 Dock 預覽

### 桌布背景

開啟 **顯示桌布背景** 後，Dock 預覽會顯示你的桌面桌布作為背景，模擬真實的 Dock 外觀。

啟用後會出現兩個額外的滑桿：

- **桌布擴展** (50–120) — 控制桌布超出 Dock 區域的擴展範圍
- **桌布模糊** (28–40) — 控制 Dock 背後桌布的模糊強度

### Dock 玻璃效果強度 (macOS 26+)

在 macOS Tahoe 及更高版本上，**Dock 玻璃效果強度** 滑桿 (0–1) 可控制預覽中 Liquid Glass Dock 框架的透明度。

## 語言

從 30 種支援的語言中選擇。變更在重新啟動應用程式後完全生效。
