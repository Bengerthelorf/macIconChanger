# API 金鑰

線上搜尋圖示需要 [macosicons.com](https://macosicons.com/) 的 API 金鑰。沒有金鑰也可以使用本機圖片。

## 取得 API 金鑰

1. 前往 [macosicons.com](https://macosicons.com/) 並註冊帳號。
2. 在帳號設定中申請 API 金鑰。
3. 複製金鑰。

![取得 API 金鑰](/images/api-key.png)

## 輸入金鑰

1. 打開 IconChanger。
2. 進入 **設定** > **進階**。
3. 在 **API Key** 中貼上金鑰。
4. 點擊 **測試連線** 驗證是否有效。

![API 金鑰設定](/images/api-key-settings.png)

## 不使用 API 金鑰

沒有 API 金鑰也可以更改圖示：

- 使用本機圖片（點擊 **從本機選擇** 或拖放圖片）
- 使用應用程式自帶的圖示（顯示在「本機」區域）

## 疑難排解

如果 API 測試失敗：
- 檢查金鑰是否正確（沒有多餘空格）
- 確認網路連線正常
- macosicons.com API 可能暫時無法使用
