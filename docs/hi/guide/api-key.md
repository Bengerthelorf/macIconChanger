# API Key

ऑनलाइन आइकन खोजने के लिए [macosicons.com](https://macosicons.com/) से एक API key आवश्यक है। इसके बिना भी आप लोकल इमेज फ़ाइलों का उपयोग कर सकते हैं।

## API Key प्राप्त करना

1. [macosicons.com](https://macosicons.com/) पर जाएं और एक अकाउंट बनाएं।
2. अपनी अकाउंट सेटिंग्स से API key का अनुरोध करें।
3. key कॉपी करें।

![API key कैसे प्राप्त करें](/images/api-key.png)

## Key दर्ज करना

1. IconChanger खोलें।
2. **Settings** > **Advanced** पर जाएं।
3. **API Key** फ़ील्ड में अपनी API key पेस्ट करें।
4. यह काम कर रही है इसकी पुष्टि के लिए **Test Connection** पर क्लिक करें।

![API key सेटिंग्स](/images/api-key-settings.png)

## बिना API Key के उपयोग

आप बिना API key के भी ऐप आइकन बदल सकते हैं:

- लोकल इमेज फ़ाइलों का उपयोग करके (**Choose from the Local** पर क्लिक करें या इमेज ड्रैग और ड्रॉप करें)
- ऐप में शामिल बिल्ट-इन आइकन का उपयोग करके ("Local" सेक्शन में दिखाए गए)

## उन्नत API सेटिंग्स

**Settings** > **Advanced** > **API Settings** में, आप API के व्यवहार को बारीकी से समायोजित कर सकते हैं:

| सेटिंग | डिफ़ॉल्ट | विवरण |
|---|---|---|
| **Retry Count** | 0 (कोई पुनः प्रयास नहीं) | विफल अनुरोध को कितनी बार पुनः प्रयास करना है (0–3) |
| **Timeout** | 15 सेकंड | प्रत्येक अनुरोध प्रयास के लिए टाइमआउट |
| **Monthly Limit** | 50 | प्रति माह अधिकतम API क्वेरी |

**Monthly Usage** काउंटर आपके वर्तमान उपयोग को दिखाता है। यह हर महीने की 1 तारीख को स्वचालित रूप से रीसेट होता है, या आप इसे मैन्युअल रूप से रीसेट कर सकते हैं।

### आइकन खोज कैश

खोज परिणामों को डिस्क पर सहेजने के लिए **Cache API Results** चालू करें। कैश किए गए परिणाम ऐप रीस्टार्ट के बाद भी बने रहते हैं, जिससे API का उपयोग कम होता है। नवीनतम परिणाम प्राप्त करने के लिए आइकन ब्राउज़ करते समय रिफ्रेश बटन का उपयोग करें।

## समस्या निवारण

यदि API टेस्ट विफल हो जाता है:
- जांचें कि आपकी key सही है (कोई अतिरिक्त स्पेस नहीं)
- अपना इंटरनेट कनेक्शन सत्यापित करें
- macosicons.com API अस्थायी रूप से अनुपलब्ध हो सकता है
