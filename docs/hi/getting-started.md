## आवश्यकताएं

- macOS 13.0 (Ventura) या बाद का संस्करण
- प्रशासक (Administrator) विशेषाधिकार (प्रारंभिक सेटअप और आइकन बदलने के लिए)

## इंस्टॉलेशन

### Homebrew (अनुशंसित)

```bash
brew install Bengerthelorf/tap/iconchanger
```

### मैनुअल डाउनलोड

1. [GitHub Releases](https://github.com/Bengerthelorf/macIconChanger/releases/latest) से नवीनतम DMG डाउनलोड करें।
2. DMG खोलें और **IconChanger** को अपने Applications फ़ोल्डर में ड्रैग करें।
3. IconChanger लॉन्च करें।

## पहला लॉन्च

पहली बार लॉन्च करने पर, IconChanger आपसे एक बार की अनुमति सेटअप पूरा करने के लिए कहेगा। ऐप को एप्लिकेशन आइकन बदलने के लिए यह आवश्यक है।

![पहली बार लॉन्च सेटअप स्क्रीन](/images/setup-prompt.png)

सेटअप बटन पर क्लिक करें और अपना प्रशासक पासवर्ड दर्ज करें। IconChanger स्वचालित रूप से आवश्यक अनुमतियां कॉन्फ़िगर करेगा (हेल्पर स्क्रिप्ट के लिए एक sudoers नियम)।

::: tip
यदि स्वचालित सेटअप विफल हो जाता है, तो मैनुअल निर्देशों के लिए [प्रारंभिक सेटअप](./setup) देखें।
:::

## अपना पहला आइकन बदलें

1. साइडबार से एक एप्लिकेशन चुनें।
2. [macOSicons.com](https://macosicons.com/) से आइकन ब्राउज़ करें या एक लोकल इमेज फ़ाइल चुनें।
3. आइकन लागू करने के लिए उस पर क्लिक करें।

![मुख्य इंटरफ़ेस](/images/main-interface.png)

बस इतना ही! ऐप आइकन तुरंत बदल जाएगा।
