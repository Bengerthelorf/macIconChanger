# Alias Aplikasi

Beberapa aplikasi memiliki nama yang tidak cocok dengan kueri pencarian ikon. Misalnya, "Adobe Photoshop 2024" mungkin tidak memberikan hasil yang baik, tetapi "Photoshop" akan memberikannya. Alias menyelesaikan masalah ini dengan memetakan nama pencarian alternatif ke suatu aplikasi.

## Mengatur Alias

1. Klik kanan aplikasi di sidebar.
2. Pilih **Set the Alias**.
3. Masukkan nama yang ramah pencarian (misalnya, `Photoshop` alih-alih `Adobe Photoshop 2024`).

![Mengatur alias](/images/set-alias.png)

Aplikasi sekarang akan menggunakan nama alias saat mencari ikon di macOSicons.com.

## Mengelola Alias

Lihat dan edit semua alias di **Settings** > **Application** > **Alias Names**.

![Pengaturan alias](/images/alias-settings.png)

## Alias Bawaan

IconChanger menyertakan beberapa alias bawaan untuk aplikasi umum:
- `PyCharm Professional Edition` → `PyCharm`
- `Discord PTB` → `Discord`

## Tips

- Gunakan nama alias yang pendek dan umum untuk hasil pencarian yang lebih baik.
- Alias disertakan saat Anda [mengekspor konfigurasi](./import-export).
