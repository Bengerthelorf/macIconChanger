# การเปลี่ยนไอคอน

## การใช้ GUI

### ค้นหาออนไลน์

1. เลือกแอปจากแถบด้านข้าง
2. เลือกดูไอคอนจาก [macOSicons.com](https://macosicons.com/) ในพื้นที่หลัก
3. ใช้ดรอปดาวน์ **Style** เพื่อกรองตามสไตล์ (เช่น Liquid Glass)
4. คลิกที่ไอคอนเพื่อนำไปใช้

![ค้นหาไอคอน](/images/search-icons.png)

### เลือกไฟล์จากเครื่อง

คลิก **Choose from the Local** (หรือกด <kbd>Cmd</kbd>+<kbd>O</kbd>) เพื่อเปิดตัวเลือกไฟล์ รูปแบบที่รองรับ: PNG, JPEG, ICNS, TIFF, HEIC, WebP, BMP, GIF, SVG

### ลากและวาง

ลากไฟล์ภาพจาก Finder ไปวางบนพื้นที่ไอคอนของแอปโดยตรง จะมีไฮไลต์สีน้ำเงินปรากฏขึ้นเพื่อยืนยันโซนวาง

![ลากและวาง](/images/drag-drop.png)

### กู้คืนไอคอนเริ่มต้น

เพื่อกู้คืนไอคอนเดิมของแอป:
- คลิกปุ่ม **Restore Default** (หรือกด <kbd>Cmd</kbd>+<kbd>Delete</kbd>)
- หรือคลิกขวาที่แอปในแถบด้านข้างแล้วเลือก **Restore Default Icon**

## หลุดจาก Squircle Jail (macOS Tahoe)

macOS 26 Tahoe บังคับให้ไอคอนแอปทั้งหมดเป็นรูปทรง squircle (สี่เหลี่ยมมุมโค้ง) แอปที่มีไอคอนไม่ตรงตามรูปแบบจะถูกย่อขนาดและวางบนพื้นหลัง squircle สีเทา

IconChanger สามารถแก้ไขปัญหานี้ได้โดยการนำไอคอนที่มาพร้อมแอปมาใช้ใหม่เป็นไอคอนกำหนดเอง ซึ่งจะข้ามการบังคับ squircle ของ macOS

### ทำทีละแอป

คลิกขวาที่แอปในแถบด้านข้างแล้วเลือก **Escape Squircle Jail**

### ทำทุกแอปพร้อมกัน

คลิกเมนู **⋯** ในแถบเครื่องมือแล้วเลือก **Escape Squircle Jail (All Apps)** ระบบจะประมวลผลแอปทั้งหมดที่ยังไม่มีไอคอนกำหนดเอง

::: tip
ไอคอนกำหนดเองที่ตั้งด้วยวิธีนี้จะ**ไม่**รองรับโหมดไอคอน Clear, Tinted หรือ Dark ของ macOS Tahoe โดยจะคงเป็นภาพนิ่ง นี่เป็นข้อจำกัดของระบบ
:::

::: info
บริการเบื้องหลังจะนำไอคอนกลับมาใช้ใหม่โดยอัตโนมัติหลังจากแอปได้รับการอัปเดต เพื่อให้ไอคอนยังคงอยู่นอก squircle jail
:::

## ไอคอนโฟลเดอร์

คุณสามารถปรับแต่งไอคอนโฟลเดอร์ได้เช่นกัน เพิ่มโฟลเดอร์ผ่าน **Settings** > **Application** > **Application Folders** หรือคลิกปุ่ม **+** ในส่วนโฟลเดอร์ของแถบด้านข้าง

เมื่อเพิ่มโฟลเดอร์แล้ว จะปรากฏในแถบด้านข้างเหมือนกับแอป คุณสามารถค้นหาไอคอน ลากและวางรูปภาพ หรือเลือกไฟล์จากเครื่อง — ขั้นตอนเดียวกับการเปลี่ยนไอคอนแอป

::: tip
ชื่อโฟลเดอร์เช่น "go" หรือ "Downloads" อาจไม่ให้ผลการค้นหาที่ดีจาก macOSicons.com ใช้[ชื่อแทน](./aliases)เพื่อตั้งชื่อที่เหมาะสำหรับการค้นหามากกว่า (เช่น ตั้งชื่อแทน "Documents" เป็น "folder")
:::

## การแคชไอคอน

เมื่อคุณใช้ไอคอนกำหนดเอง ไอคอนจะถูกแคชโดยอัตโนมัติ ซึ่งหมายความว่า:
- ไอคอนกำหนดเองของคุณสามารถกู้คืนได้หลังจากแอปได้รับการอัปเดต
- บริการเบื้องหลังสามารถนำไอคอนกลับมาใช้ใหม่ตามกำหนดเวลา
- คุณสามารถส่งออกและนำเข้าการตั้งค่าไอคอนของคุณได้

จัดการไอคอนที่แคชไว้ได้ใน **Settings** > **Icon Cache**

## แป้นพิมพ์ลัด

| แป้นพิมพ์ลัด | การทำงาน |
|---|---|
| <kbd>Cmd</kbd>+<kbd>O</kbd> | เลือกไฟล์ไอคอนจากเครื่อง |
| <kbd>Cmd</kbd>+<kbd>Delete</kbd> | กู้คืนไอคอนเริ่มต้น |
| <kbd>Cmd</kbd>+<kbd>R</kbd> | รีเฟรชการแสดงผลไอคอน |

## เคล็ดลับ

- หากไม่พบไอคอนสำหรับแอป ลอง[ตั้งชื่อแทน](./aliases)ด้วยชื่อที่ง่ายกว่า
- ตัวนับ (เช่น "12/15") แสดงจำนวนไอคอนที่โหลดสำเร็จจากทั้งหมดที่พบ
- ไอคอนจะถูกเรียงตามความนิยม (จำนวนดาวน์โหลด) เป็นค่าเริ่มต้น
