# API Key

จำเป็นต้องมี API key จาก [macosicons.com](https://macosicons.com/) เพื่อค้นหาไอคอนออนไลน์ หากไม่มี คุณยังสามารถใช้ไฟล์ภาพจากเครื่องได้

## การขอ API Key

1. ไปที่ [macosicons.com](https://macosicons.com/) แล้วสร้างบัญชี
2. ขอ API key จากหน้าการตั้งค่าบัญชีของคุณ
3. คัดลอกคีย์

![วิธีขอ API key](/images/api-key.png)

## การป้อนคีย์

1. เปิด IconChanger
2. ไปที่ **Settings** > **Advanced**
3. วาง API key ของคุณในช่อง **API Key**
4. คลิก **Test Connection** เพื่อตรวจสอบว่าใช้งานได้

![การตั้งค่า API key](/images/api-key-settings.png)

## การใช้งานโดยไม่มี API Key

คุณยังสามารถเปลี่ยนไอคอนแอปได้โดยไม่ต้องมี API key โดย:

- ใช้ไฟล์ภาพจากเครื่อง (คลิก **Choose from the Local** หรือลากแล้ววางภาพ)
- ใช้ไอคอนที่มีอยู่ในตัวแอปเอง (แสดงในส่วน "Local")

## การแก้ไขปัญหา

หากการทดสอบ API ล้มเหลว:
- ตรวจสอบว่าคีย์ของคุณถูกต้อง (ไม่มีช่องว่างเกิน)
- ตรวจสอบการเชื่อมต่ออินเทอร์เน็ตของคุณ
- API ของ macosicons.com อาจไม่พร้อมใช้งานชั่วคราว
