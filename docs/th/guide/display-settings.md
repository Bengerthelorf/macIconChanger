# การตั้งค่าการแสดงผล

กำหนดค่าลักษณะที่ปรากฏของ IconChanger ใน **Settings** > **Display**

## ลักษณะที่ปรากฏ

เลือกระหว่างโหมด **ระบบ**, **สว่าง** หรือ **มืด** ตัวเลือกระบบจะตามการตั้งค่าลักษณะที่ปรากฏของ macOS

### ป้ายไอคอนที่กำหนดเอง

เปิด **แสดงป้ายไอคอนที่กำหนดเอง** เพื่อแสดงเครื่องหมายถูกสีเขียวบนแอปในแถบด้านข้างที่ใช้ไอคอนที่กำหนดเอง

## ตัวอย่าง Dock

ตัวอย่าง Dock แสดงตัวอย่างแบบเรียลไทม์ว่าไอคอนที่กำหนดเองของคุณจะมีลักษณะอย่างไรใน Dock ของ macOS

### โหมดตัวอย่าง

- **เฉพาะเมื่อกรอง Dock** — แสดงตัวอย่าง Dock เฉพาะเมื่อคุณกรองตามแอป Dock
- **เสมอ** — แสดงตัวอย่าง Dock ที่ด้านบนของพื้นที่รายละเอียดเสมอ

### วอลเปเปอร์พื้นหลัง

เปิด **แสดงวอลเปเปอร์พื้นหลัง** เพื่อแสดงวอลเปเปอร์เดสก์ท็อปของคุณด้านหลังตัวอย่าง Dock เพื่อจำลองลักษณะ Dock จริง

เมื่อเปิดใช้งาน จะปรากฏแถบเลื่อนเพิ่มเติมสองตัว:

- **การขยายวอลเปเปอร์** (50–120) — ควบคุมว่าวอลเปเปอร์ขยายออกไปนอกพื้นที่ Dock มากเพียงใด
- **ความเบลอของวอลเปเปอร์** (28–40) — ควบคุมความเข้มของการเบลอวอลเปเปอร์ด้านหลัง Dock

### ความเข้มกระจก Dock (macOS 26+)

บน macOS Tahoe ขึ้นไป แถบเลื่อน **ความเข้มกระจก Dock** (0–1) ควบคุมความโปร่งแสงของกรอบ Liquid Glass Dock ในตัวอย่าง

## ภาษา

เลือกจาก 30 ภาษาที่รองรับ การเปลี่ยนแปลงจะมีผลสมบูรณ์หลังจากรีสตาร์ทแอป
