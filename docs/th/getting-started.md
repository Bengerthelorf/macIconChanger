## ข้อกำหนด

- macOS 13.0 (Ventura) หรือใหม่กว่า
- สิทธิ์ผู้ดูแลระบบ (สำหรับการตั้งค่าเริ่มต้นและการเปลี่ยนไอคอน)

## การติดตั้ง

### Homebrew (แนะนำ)

```bash
brew install Bengerthelorf/tap/iconchanger
```

### ดาวน์โหลดด้วยตนเอง

1. ดาวน์โหลดไฟล์ DMG ล่าสุดจาก [GitHub Releases](https://github.com/Bengerthelorf/macIconChanger/releases/latest)
2. เปิดไฟล์ DMG แล้วลาก **IconChanger** ไปยังโฟลเดอร์ Applications
3. เปิด IconChanger

## การเปิดใช้งานครั้งแรก

เมื่อเปิดใช้งานครั้งแรก IconChanger จะแจ้งให้คุณทำการตั้งค่าสิทธิ์ครั้งเดียว ซึ่งจำเป็นสำหรับให้แอปสามารถเปลี่ยนไอคอนของแอปพลิเคชันได้

![หน้าจอตั้งค่าเมื่อเปิดครั้งแรก](/images/setup-prompt.png)

คลิกปุ่มตั้งค่าแล้วป้อนรหัสผ่านผู้ดูแลระบบ IconChanger จะกำหนดค่าสิทธิ์ที่จำเป็นโดยอัตโนมัติ (กฎ sudoers สำหรับ helper script)

::: tip
หากการตั้งค่าอัตโนมัติล้มเหลว ดู[การตั้งค่าเบื้องต้น](./setup)สำหรับคำแนะนำในการตั้งค่าด้วยตนเอง
:::

## เปลี่ยนไอคอนแรกของคุณ

1. เลือกแอปพลิเคชันจากแถบด้านข้าง
2. เลือกดูไอคอนจาก [macOSicons.com](https://macosicons.com/) หรือเลือกไฟล์ภาพจากเครื่อง
3. คลิกที่ไอคอนเพื่อนำไปใช้

![หน้าจอหลัก](/images/main-interface.png)

เท่านี้ก็เรียบร้อย! ไอคอนของแอปจะถูกเปลี่ยนทันที
