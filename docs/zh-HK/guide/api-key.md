# API 密鑰

線上搜尋圖示需要 [macosicons.com](https://macosicons.com/) 的 API 密鑰。沒有密鑰也可以使用本機圖片。

## 取得 API 密鑰

1. 前往 [macosicons.com](https://macosicons.com/) 並註冊帳號。
2. 在帳號設定中申請 API 密鑰。
3. 複製密鑰。

![取得 API 密鑰](/images/api-key.png)

## 輸入密鑰

1. 打開 IconChanger。
2. 進入 **設定** > **進階**。
3. 在 **API Key** 中貼上密鑰。
4. 點擊 **測試連接** 驗證是否有效。

![API 密鑰設定](/images/api-key-settings.png)

## 不使用 API 密鑰

沒有 API 密鑰也可以更改圖示：

- 使用本機圖片（點擊 **從本機選擇** 或拖放圖片）
- 使用應用程式自帶的圖示（顯示在「本機」區域）

## 進階 API 設定

在 **設定** > **進階** > **API 設定** 中，你可以微調 API 行為：

| 設定 | 預設值 | 說明 |
|---|---|---|
| **重試次數** | 0（不重試） | 請求失敗後重試嘅次數（0–3） |
| **逾時時間** | 15 秒 | 每個請求嘅逾時時間 |
| **每月限額** | 50 | 每月最大 API 查詢次數 |

**每月使用量** 計數器顯示目前嘅使用情況。佢會喺每月 1 日自動重置，亦可以手動重置。

### 圖示搜尋快取

開啟 **快取 API 結果** 可將搜尋結果儲存到磁碟。快取結果喺應用程式重新啟動後仍然保留，從而減少 API 使用量。瀏覽圖示時使用重新整理按鈕可取得最新結果。

## 疑難排解

如果 API 測試失敗：
- 檢查密鑰是否正確（沒有多餘空格）
- 確認網絡連接正常
- macosicons.com API 可能暫時無法使用
