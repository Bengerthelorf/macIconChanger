# 更改圖示

## 使用界面

### 線上搜尋

1. 從側邊欄選擇應用程式。
2. 在主區域瀏覽來自 [macOSicons.com](https://macosicons.com/) 的圖示。
3. 使用 **Style** 下拉選單按風格篩選（如 Liquid Glass）。
4. 點擊圖示即可應用。

![搜尋圖示](/images/search-icons.png)

### 選擇本機檔案

點擊 **從本機選擇**（或按 <kbd>Cmd</kbd>+<kbd>O</kbd>）打開檔案選擇器。支援的格式：PNG、JPEG、ICNS、TIFF、HEIC、WebP、BMP、GIF、SVG。

### 拖放

直接從 Finder 拖放圖片檔案到圖示區域。拖放時會顯示藍色高亮提示確認拖放區域。

![拖放](/images/drag-drop.png)

### 還原預設圖示

還原應用程式的原始圖示：
- 點擊 **還原預設** 按鈕（或按 <kbd>Cmd</kbd>+<kbd>Delete</kbd>）
- 或右鍵側邊欄中的應用程式，選擇 **還原預設圖示**

## 逃出圓角矩形監獄（macOS Tahoe）

macOS 26 Tahoe 強制所有應用程式圖示顯示為圓角矩形（squircle）形狀。不符合標準的圖示會被縮小並放在灰色圓角矩形背景上。

IconChanger 可以透過將應用程式自身的圖示重新設定為自訂圖示來解決這個問題，因為自訂圖示不受系統圓角矩形強制裁切的影響。

### 單一應用程式

右鍵側邊欄中的應用程式，選擇 **逃出圓角矩形監獄**。

### 批次處理

點擊工具列的 **⋯** 選單，選擇 **逃出圓角矩形監獄（所有應用程式）**。這會處理所有尚未設定自訂圖示的應用程式。

::: tip
透過這種方式設定的自訂圖示**不支援** macOS Tahoe 的 Clear（透明）、Tinted（著色）或 Dark（深色）圖示模式——它們保持為靜態圖片。這是系統限制。
:::

::: info
背景服務會在應用程式更新後自動重新應用圖示，使它們不會再被關進圓角矩形監獄。
:::

## 圖示緩存

應用自訂圖示時會自動緩存，這意味着：
- 應用程式更新後可以還原自訂圖示
- 背景服務可以定時重新應用
- 可以導出和導入圖示設定

在 **設定** > **圖示緩存** 中管理緩存的圖示。

## 快速鍵

| 快速鍵 | 動作 |
|---|---|
| <kbd>Cmd</kbd>+<kbd>O</kbd> | 選擇本機圖示檔案 |
| <kbd>Cmd</kbd>+<kbd>Delete</kbd> | 還原預設圖示 |
| <kbd>Cmd</kbd>+<kbd>R</kbd> | 刷新圖示顯示 |

## 提示

- 如果某個應用程式搜尋不到圖示，試試[設定別名](./aliases)，使用更簡短的名稱。
- 計數器（如 "12/15"）顯示成功載入的圖示數量。
- 圖示預設按熱門程度（下載量）排序。
