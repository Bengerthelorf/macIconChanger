**설정** > **디스플레이**에서 IconChanger의 시각적 모양을 구성할 수 있습니다.

## 외관

**시스템**, **라이트**, **다크** 모드 중에서 선택합니다. 시스템 옵션은 macOS 외관 설정을 따릅니다.

### 사용자 지정 아이콘 배지

**사용자 지정 아이콘 배지 표시**를 활성화하면, 사이드바에서 사용자 지정 아이콘이 적용된 앱에 녹색 체크 표시가 나타납니다.

## Dock 미리보기

Dock 미리보기는 사용자 지정 아이콘이 macOS Dock에서 어떻게 보이는지 실시간으로 미리 보여줍니다.

### 미리보기 모드

- **Dock 필터 시에만** — Dock 앱으로 필터링할 때만 Dock 미리보기를 표시합니다
- **항상** — 상세 영역 상단에 항상 Dock 미리보기를 표시합니다

### 배경화면 배경

**배경화면 배경 표시**를 활성화하면, Dock 미리보기 뒤에 데스크톱 배경화면이 표시되어 실제 Dock 모습을 재현합니다.

활성화하면 두 개의 추가 슬라이더가 나타납니다:

- **배경화면 확장** (50–120) — 배경화면이 Dock 영역 너머로 확장되는 범위를 제어합니다
- **배경화면 블러** (28–40) — Dock 뒤 배경화면의 블러 강도를 제어합니다

### Dock 글래스 강도 (macOS 26+)

macOS Tahoe 이상에서는 **Dock 글래스 강도** 슬라이더 (0–1)로 미리보기에서 Liquid Glass Dock 프레임의 투명도를 제어할 수 있습니다.

## 언어

30개의 지원 언어 중에서 선택할 수 있습니다. 변경 사항은 앱을 재시작한 후 완전히 적용됩니다.
