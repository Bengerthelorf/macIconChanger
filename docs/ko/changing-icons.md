## GUI 사용

### 온라인 검색

1. 사이드바에서 앱을 선택합니다.
2. 메인 영역에서 [macOSicons.com](https://macosicons.com/)의 아이콘을 탐색합니다.
3. **스타일** 드롭다운을 사용하여 스타일별로 필터링합니다 (예: Liquid Glass).
4. 아이콘을 클릭하여 적용합니다.

![아이콘 검색](/images/search-icons.png)

### 로컬 파일 선택

**로컬에서 선택**을 클릭하거나 <kbd>Cmd</kbd>+<kbd>O</kbd>를 눌러 파일 선택기를 엽니다. 지원 형식: PNG, JPEG, ICNS, TIFF, HEIC, WebP, BMP, GIF, SVG.

### 드래그 앤 드롭

Finder에서 이미지 파일을 앱의 아이콘 영역으로 직접 드래그합니다. 파란색 강조 표시가 나타나 드롭 영역을 확인할 수 있습니다.

![드래그 앤 드롭](/images/drag-drop.png)

### 기본 아이콘 복원

앱의 원래 아이콘을 복원하려면:
- **기본값 복원** 버튼을 클릭합니다 (또는 <kbd>Cmd</kbd>+<kbd>Delete</kbd>를 누릅니다)
- 또는 사이드바에서 앱을 우클릭하고 **기본 아이콘 복원**을 선택합니다

## Squircle Jail 탈출 (macOS Tahoe)

macOS 26 Tahoe는 모든 앱 아이콘을 스퀘클(둥근 사각형) 모양으로 강제합니다. 규격에 맞지 않는 아이콘을 가진 앱은 아이콘이 축소되어 회색 스퀘클 배경 위에 배치됩니다.

IconChanger는 앱의 자체 번들 아이콘을 커스텀 아이콘으로 다시 적용하여 이 문제를 해결할 수 있습니다. 커스텀 아이콘은 macOS의 스퀘클 강제를 우회합니다.

### 개별 앱

사이드바에서 앱을 우클릭하고 **Squircle Jail 탈출**을 선택합니다.

### 모든 앱 일괄 적용

도구 모음의 **⋯** 메뉴를 클릭하고 **Squircle Jail 탈출 (모든 앱)**을 선택합니다. 이미 커스텀 아이콘이 설정되지 않은 모든 앱을 처리합니다.

::: tip
이 방식으로 설정된 커스텀 아이콘은 macOS Tahoe의 Clear, Tinted 또는 Dark 아이콘 모드를 지원하지 **않습니다**. 아이콘은 정적 상태로 유지됩니다. 이는 시스템의 제한 사항입니다.
:::

::: info
백그라운드 서비스가 앱 업데이트 후 자동으로 아이콘을 다시 적용하여 Squircle Jail 상태를 방지합니다.
:::

## 폴더 아이콘

폴더 아이콘도 커스터마이즈할 수 있습니다. **설정** > **응용 프로그램** > **응용 프로그램 폴더**에서 폴더를 추가하거나, 사이드바의 폴더 섹션에서 **+** 버튼을 클릭하세요.

폴더가 추가되면 앱처럼 사이드바에 표시됩니다. 아이콘 검색, 이미지 드래그 앤 드롭, 로컬 파일 선택 등 앱 아이콘을 변경할 때와 동일한 워크플로우로 사용할 수 있습니다.

::: tip
"go"나 "Downloads" 같은 폴더 이름은 macOSicons.com에서 좋은 검색 결과를 얻지 못할 수 있습니다. [별칭](./aliases)을 사용하여 더 검색에 적합한 이름을 설정하세요 (예: "Documents"의 별칭을 "folder"로 설정).
:::

## 아이콘 캐싱

커스텀 아이콘을 적용하면 자동으로 캐시됩니다. 이를 통해:
- 앱 업데이트 후 커스텀 아이콘을 복원할 수 있습니다
- 백그라운드 서비스가 예약에 따라 아이콘을 다시 적용할 수 있습니다
- 아이콘 설정을 내보내고 가져올 수 있습니다

**설정** > **아이콘 캐시**에서 캐시된 아이콘을 관리할 수 있습니다.

## 키보드 단축키

| 단축키 | 동작 |
|---|---|
| <kbd>Cmd</kbd>+<kbd>O</kbd> | 로컬 아이콘 파일 선택 |
| <kbd>Cmd</kbd>+<kbd>Delete</kbd> | 기본 아이콘 복원 |
| <kbd>Cmd</kbd>+<kbd>R</kbd> | 아이콘 표시 새로고침 |

## 팁

- 앱에 대한 아이콘이 검색되지 않으면 더 간단한 이름으로 [별칭 설정](./aliases)을 시도해 보세요.
- 카운터 (예: "12/15")는 전체 검색 결과 중 성공적으로 로드된 아이콘 수를 나타냅니다.
- 아이콘은 기본적으로 인기도(다운로드 수) 순으로 정렬됩니다.
