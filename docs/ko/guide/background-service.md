# 백그라운드 서비스

백그라운드 서비스는 앱 업데이트나 시스템 변경 후에도 커스텀 아이콘을 유지합니다.

## 활성화

**설정** > **백그라운드**로 이동하여 **백그라운드에서 실행**을 토글합니다.

활성화하면 창을 닫아도 IconChanger가 계속 실행됩니다. 메뉴 바 또는 Dock에서 접근할 수 있습니다.

## 기능

### 예약 복원

캐시된 모든 커스텀 아이콘을 정기적으로 자동 복원합니다.

- **예약에 따라 아이콘 복원**을 토글합니다
- 간격을 선택합니다: 매시간, 3시간, 6시간, 12시간, 매일 또는 사용자 지정 간격
- 설정에서 마지막 복원 시점과 다음 복원 시점을 확인할 수 있습니다

### 앱 업데이트 감지

앱이 업데이트되면 이를 감지하고 커스텀 아이콘을 자동으로 다시 적용합니다.

- **앱 업데이트 시 아이콘 복원**을 토글합니다
- 업데이트 확인 간격을 설정합니다 (5분마다에서 2시간마다 또는 사용자 지정)

### 앱 표시

백그라운드에서 실행 중일 때 IconChanger가 표시되는 위치를 제어합니다:

- **메뉴 바에 표시** — 메뉴 바에 아이콘을 추가합니다
- **Dock에 표시** — Dock에 앱을 유지합니다

이 중 하나 이상이 활성화되어 있어야 합니다.

### 로그인 시 실행

Mac에 로그인할 때 IconChanger를 자동으로 시작합니다.

- **메인 창 열기** — 메인 창이 표시되며 정상적으로 실행됩니다
- **숨김 상태로 시작** — 백그라운드에서 조용히 실행됩니다 ("백그라운드에서 실행"이 활성화되어 있어야 합니다)

::: info
"숨김 상태로 시작"은 로그인 시 실행에만 적용됩니다. 앱을 수동으로 열면 항상 메인 창이 표시됩니다.
:::

## 서비스 상태

백그라운드 서비스가 활성화되어 있으면 설정 페이지에 다음이 표시됩니다:
- **서비스 상태** — 서비스가 실행 중인지 여부
- **캐시된 아이콘** — 복원할 준비가 된 아이콘 수
