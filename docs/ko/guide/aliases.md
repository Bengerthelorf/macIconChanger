# 앱 별칭

일부 앱은 이름이 아이콘 검색 쿼리와 잘 맞지 않습니다. 예를 들어, "Adobe Photoshop 2024"로는 좋은 결과를 얻기 어렵지만 "Photoshop"으로 검색하면 원하는 결과를 얻을 수 있습니다. 별칭은 앱에 대체 검색 이름을 매핑하여 이 문제를 해결합니다.

## 별칭 설정

1. 사이드바에서 앱을 우클릭합니다.
2. **별칭 설정**을 선택합니다.
3. 검색에 적합한 이름을 입력합니다 (예: `Adobe Photoshop 2024` 대신 `Photoshop`).

![별칭 설정](/images/set-alias.png)

이제 macOSicons.com에서 아이콘을 검색할 때 별칭 이름이 사용됩니다.

## 별칭 관리

**설정** > **응용 프로그램** > **별칭 이름**에서 모든 별칭을 확인하고 편집할 수 있습니다.

![별칭 설정](/images/alias-settings.png)

## 내장 별칭

IconChanger는 일부 일반적인 앱에 대한 내장 별칭을 포함하고 있습니다:
- `PyCharm Professional Edition` → `PyCharm`
- `Discord PTB` → `Discord`

## 팁

- 더 나은 검색 결과를 위해 별칭 이름은 짧고 일반적인 이름으로 유지하세요.
- 별칭은 [설정 내보내기](./import-export) 시 함께 포함됩니다.
