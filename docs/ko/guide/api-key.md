# API 키

온라인에서 아이콘을 검색하려면 [macosicons.com](https://macosicons.com/)의 API 키가 필요합니다. API 키가 없어도 로컬 이미지 파일을 사용할 수 있습니다.

## API 키 발급

1. [macosicons.com](https://macosicons.com/)을 방문하여 계정을 만듭니다.
2. 계정 설정에서 API 키를 요청합니다.
3. 키를 복사합니다.

![API 키 발급 방법](/images/api-key.png)

## 키 입력

1. IconChanger를 엽니다.
2. **설정** > **고급**으로 이동합니다.
3. **API Key** 필드에 API 키를 붙여넣습니다.
4. **연결 테스트**를 클릭하여 정상적으로 작동하는지 확인합니다.

![API 키 설정](/images/api-key-settings.png)

## API 키 없이 사용

API 키 없이도 다음과 같은 방법으로 앱 아이콘을 변경할 수 있습니다:

- 로컬 이미지 파일 사용 (**로컬에서 선택** 클릭 또는 이미지 드래그 앤 드롭)
- 앱에 내장된 아이콘 사용 ("로컬" 섹션에 표시됨)

## 문제 해결

API 테스트에 실패하는 경우:
- 키가 올바른지 확인합니다 (공백이 포함되지 않았는지 확인)
- 인터넷 연결을 확인합니다
- macosicons.com API가 일시적으로 사용 불가능할 수 있습니다
