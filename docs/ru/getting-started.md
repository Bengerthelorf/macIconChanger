## Требования

- macOS 13.0 (Ventura) или новее
- Права администратора (для первоначальной настройки и смены иконок)

## Установка

### Homebrew (рекомендуется)

```bash
brew install Bengerthelorf/tap/iconchanger
```

### Ручная загрузка

1. Скачайте последнюю версию DMG с [GitHub Releases](https://github.com/Bengerthelorf/macIconChanger/releases/latest).
2. Откройте DMG и перетащите **IconChanger** в папку «Программы».
3. Запустите IconChanger.

## Первый запуск

При первом запуске IconChanger предложит выполнить однократную настройку разрешений. Это необходимо для того, чтобы приложение могло изменять иконки.

![Экран первоначальной настройки](/images/setup-prompt.png)

Нажмите кнопку настройки и введите пароль администратора. IconChanger автоматически настроит необходимые разрешения (правило sudoers для вспомогательного скрипта).

::: tip
Если автоматическая настройка не сработала, см. раздел [Первоначальная настройка](./setup) для ручной настройки.
:::

## Смена первой иконки

1. Выберите приложение в боковой панели.
2. Найдите иконки на [macOSicons.com](https://macosicons.com/) или выберите локальный файл изображения.
3. Нажмите на иконку, чтобы применить её.

![Основной интерфейс](/images/main-interface.png)

Готово! Иконка приложения будет изменена немедленно.
