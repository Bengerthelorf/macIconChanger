# Смена иконок

## Использование графического интерфейса

### Поиск онлайн

1. Выберите приложение в боковой панели.
2. Просмотрите иконки с [macOSicons.com](https://macosicons.com/) в основной области.
3. Используйте выпадающий список **Style** для фильтрации по стилю (например, Liquid Glass).
4. Нажмите на иконку, чтобы применить её.

![Поиск иконок](/images/search-icons.png)

### Выбор локального файла

Нажмите **Choose from the Local** (или <kbd>Cmd</kbd>+<kbd>O</kbd>), чтобы открыть окно выбора файла. Поддерживаемые форматы: PNG, JPEG, ICNS, TIFF, HEIC, WebP, BMP, GIF, SVG.

### Перетаскивание

Перетащите файл изображения из Finder прямо в область иконки приложения. Синяя подсветка подтвердит зону для перетаскивания.

![Перетаскивание](/images/drag-drop.png)

### Восстановление иконки по умолчанию

Чтобы восстановить оригинальную иконку приложения:
- Нажмите кнопку **Restore Default** (или <kbd>Cmd</kbd>+<kbd>Delete</kbd>)
- Или щёлкните правой кнопкой мыши по приложению в боковой панели и выберите **Restore Default Icon**

## Escape Squircle Jail (macOS Tahoe)

macOS 26 Tahoe принудительно придаёт всем иконкам приложений форму сквиркла (скруглённого квадрата). Приложения с несоответствующими иконками уменьшаются и размещаются на сером фоне в форме сквиркла.

IconChanger может исправить это, повторно применив встроенную иконку приложения как пользовательскую, что позволяет обойти принудительное применение формы сквиркла в macOS.

### Для отдельного приложения

Щёлкните правой кнопкой мыши по приложению в боковой панели и выберите **Escape Squircle Jail**.

### Для всех приложений сразу

Нажмите меню **⋯** на панели инструментов и выберите **Escape Squircle Jail (All Apps)**. Это обработает все приложения, у которых ещё нет пользовательских иконок.

::: tip
Пользовательские иконки, установленные таким способом, **не** поддерживают режимы иконок Clear, Tinted и Dark в macOS Tahoe — они остаются статичными. Это системное ограничение.
:::

::: info
Фоновый сервис автоматически повторно применит иконки после обновлений приложений, сохраняя их от принудительной формы сквиркла.
:::

## Кэширование иконок

При применении пользовательской иконки она автоматически кэшируется. Это означает:
- Ваши пользовательские иконки могут быть восстановлены после обновлений приложений
- Фоновый сервис может повторно применять их по расписанию
- Вы можете экспортировать и импортировать конфигурации иконок

Управляйте кэшированными иконками в **Settings** > **Icon Cache**.

## Сочетания клавиш

| Сочетание | Действие |
|---|---|
| <kbd>Cmd</kbd>+<kbd>O</kbd> | Выбрать локальный файл иконки |
| <kbd>Cmd</kbd>+<kbd>Delete</kbd> | Восстановить иконку по умолчанию |
| <kbd>Cmd</kbd>+<kbd>R</kbd> | Обновить отображение иконок |

## Советы

- Если для приложения не найдены иконки, попробуйте [задать псевдоним](./aliases) с более простым именем.
- Счётчик (например, «12/15») показывает, сколько иконок было успешно загружено из общего числа найденных.
- По умолчанию иконки сортируются по популярности (количеству скачиваний).
