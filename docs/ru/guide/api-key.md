# API-ключ

Для поиска иконок онлайн необходим API-ключ от [macosicons.com](https://macosicons.com/). Без него вы по-прежнему можете использовать локальные файлы изображений.

## Получение API-ключа

1. Перейдите на [macosicons.com](https://macosicons.com/) и создайте учётную запись.
2. Запросите API-ключ в настройках вашей учётной записи.
3. Скопируйте ключ.

![Как получить API-ключ](/images/api-key.png)

## Ввод ключа

1. Откройте IconChanger.
2. Перейдите в **Settings** > **Advanced**.
3. Вставьте API-ключ в поле **API Key**.
4. Нажмите **Test Connection**, чтобы проверить работоспособность.

![Настройки API-ключа](/images/api-key-settings.png)

## Использование без API-ключа

Вы можете менять иконки приложений без API-ключа следующими способами:

- Использование локальных файлов изображений (нажмите **Choose from the Local** или перетащите изображение)
- Использование иконок, встроенных в само приложение (отображаются в разделе «Local»)

## Устранение неполадок

Если проверка API-ключа не прошла:
- Убедитесь, что ключ введён правильно (без лишних пробелов)
- Проверьте подключение к интернету
- API macosicons.com может быть временно недоступен
