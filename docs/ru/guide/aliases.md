# Псевдонимы приложений

Некоторые приложения имеют названия, которые плохо подходят для поисковых запросов иконок. Например, «Adobe Photoshop 2024» может не дать хороших результатов, тогда как «Photoshop» — даст. Псевдонимы решают эту проблему, сопоставляя альтернативное имя для поиска с приложением.

## Установка псевдонима

1. Щёлкните правой кнопкой мыши по приложению в боковой панели.
2. Выберите **Set the Alias**.
3. Введите удобное для поиска имя (например, `Photoshop` вместо `Adobe Photoshop 2024`).

![Установка псевдонима](/images/set-alias.png)

Теперь приложение будет использовать псевдоним при поиске иконок на macOSicons.com.

## Управление псевдонимами

Просматривайте и редактируйте все псевдонимы в **Settings** > **Application** > **Alias Names**.

![Настройки псевдонимов](/images/alias-settings.png)

## Встроенные псевдонимы

IconChanger включает несколько встроенных псевдонимов для популярных приложений:
- `PyCharm Professional Edition` → `PyCharm`
- `Discord PTB` → `Discord`

## Советы

- Используйте короткие и общие имена псевдонимов для лучших результатов поиска.
- Псевдонимы включаются при [экспорте конфигурации](./import-export).
