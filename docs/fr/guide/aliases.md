# Alias d'applications

Certaines applications ont des noms qui ne correspondent pas bien aux requetes de recherche d'icones. Par exemple, "Adobe Photoshop 2024" pourrait ne pas donner de bons resultats, mais "Photoshop" le ferait. Les alias resolvent ce probleme en associant un nom de recherche alternatif a une application.

## Definir un alias

1. Faites un clic droit sur une application dans la barre laterale.
2. Selectionnez **Set the Alias**.
3. Saisissez un nom adapte a la recherche (par ex. `Photoshop` au lieu de `Adobe Photoshop 2024`).

![Definir un alias](/images/set-alias.png)

L'application utilisera desormais le nom de l'alias lors de la recherche d'icones sur macOSicons.com.

## Gerer les alias

Consultez et modifiez tous les alias dans **Settings** > **Application** > **Alias Names**.

![Parametres des alias](/images/alias-settings.png)

## Alias integres

IconChanger inclut des alias integres pour les applications courantes :
- `PyCharm Professional Edition` -> `PyCharm`
- `Discord PTB` -> `Discord`

## Conseils

- Privilegiez des noms d'alias courts et generiques pour de meilleurs resultats de recherche.
- Les alias sont inclus lorsque vous [exportez votre configuration](./import-export).
