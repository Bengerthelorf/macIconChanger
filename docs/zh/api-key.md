在线搜索图标需要 [macosicons.com](https://macosicons.com/) 的 API 密钥。没有密钥也可以使用本地图片。

## 获取 API 密钥

1. 访问 [macosicons.com](https://macosicons.com/) 并注册账号。
2. 在账号设置中申请 API 密钥。
3. 复制密钥。

![获取 API 密钥](/images/api-key.png)

## 输入密钥

1. 打开 IconChanger。
2. 进入 **设置** > **高级**。
3. 在 **API Key** 中粘贴密钥。
4. 点击 **测试连接** 验证是否有效。

![API 密钥设置](/images/api-key-settings.png)

## 不使用 API 密钥

没有 API 密钥也可以更改图标：

- 使用本地图片（点击 **从本地选择** 或拖放图片）
- 使用应用自带的图标（显示在"本地"区域）

## 高级 API 设置

在 **设置** > **高级** > **API 设置** 中，你可以微调 API 行为：

| 设置 | 默认值 | 说明 |
|---|---|---|
| **重试次数** | 0（不重试） | 请求失败后重试的次数（0–3） |
| **超时时间** | 15 秒 | 每个请求的超时时间 |
| **每月限额** | 50 | 每月最大 API 查询次数 |

**每月使用量** 计数器显示当前使用情况。它会在每月 1 日自动重置，也可以手动重置。

### 图标搜索缓存

开启 **缓存 API 结果** 可将搜索结果保存到磁盘。缓存结果在应用重启后仍然保留，从而减少 API 使用量。浏览图标时使用刷新按钮可获取最新结果。

## 故障排除

如果 API 测试失败：
- 检查密钥是否正确（没有多余空格）
- 确认网络连接正常
- macosicons.com API 可能暂时不可用
