# 更改图标

## 使用界面

### 在线搜索

1. 从侧边栏选择应用。
2. 在主区域浏览来自 [macOSicons.com](https://macosicons.com/) 的图标。
3. 使用 **Style** 下拉菜单按风格筛选（如 Liquid Glass）。
4. 点击图标即可应用。

![搜索图标](/images/search-icons.png)

### 选择本地文件

点击 **从本地选择**（或按 <kbd>Cmd</kbd>+<kbd>O</kbd>）打开文件选择器。支持的格式：PNG、JPEG、ICNS、TIFF、HEIC、WebP、BMP、GIF、SVG。

### 拖放

直接从访达拖放图片文件到图标区域。拖放时会显示蓝色高亮确认拖放区域。

![拖放](/images/drag-drop.png)

### 恢复默认图标

恢复应用的原始图标：
- 点击 **恢复默认** 按钮（或按 <kbd>Cmd</kbd>+<kbd>Delete</kbd>）
- 或右键侧边栏中的应用，选择 **恢复默认图标**

## 逃出圆角矩形监狱（macOS Tahoe）

macOS 26 Tahoe 强制所有应用图标显示为圆角矩形（squircle）形状。不符合标准的图标会被缩小并放在灰色圆角矩形背景上。

IconChanger 可以通过将应用自身的图标重新设置为自定义图标来解决这个问题，因为自定义图标不受系统圆角矩形强制裁剪的影响。

### 单个应用

右键侧边栏中的应用，选择 **逃出圆角矩形监狱**。

### 批量处理

点击工具栏的 **⋯** 菜单，选择 **逃出圆角矩形监狱（所有应用）**。这会处理所有尚未设置自定义图标的应用。

::: tip
通过这种方式设置的自定义图标**不支持** macOS Tahoe 的 Clear（透明）、Tinted（着色）或 Dark（深色）图标模式——它们保持为静态图片。这是系统限制。
:::

::: info
后台服务会在应用更新后自动重新应用图标，使它们不会再被关进圆角矩形监狱。
:::

## 文件夹图标

你也可以自定义文件夹图标。通过 **设置** > **应用** > **应用文件夹** 添加文件夹，或者点击侧边栏文件夹区域的 **+** 按钮。

文件夹添加后会像应用一样显示在侧边栏中。你可以搜索图标、拖放图片或选择本地文件——与更改应用图标的流程完全相同。

::: tip
像 "go" 或 "Downloads" 这样的文件夹名称可能在 macOSicons.com 上搜索不到好的结果。使用[别名](./aliases)设置一个更便于搜索的名称（例如，将 "Documents" 的别名设为 "folder"）。
:::

## 图标缓存

应用自定义图标时会自动缓存，这意味着：
- 应用更新后可以恢复自定义图标
- 后台服务可以按计划重新应用
- 可以导出和导入图标配置

在 **设置** > **图标缓存** 中管理缓存的图标。

## 快捷键

| 快捷键 | 操作 |
|---|---|
| <kbd>Cmd</kbd>+<kbd>O</kbd> | 选择本地图标文件 |
| <kbd>Cmd</kbd>+<kbd>Delete</kbd> | 恢复默认图标 |
| <kbd>Cmd</kbd>+<kbd>R</kbd> | 刷新图标显示 |

## 提示

- 如果某个应用搜索不到图标，试试[设置别名](./aliases)，使用更简短的名称。
- 计数器（如 "12/15"）显示成功加载的图标数量。
- 图标默认按人气（下载量）排序。
