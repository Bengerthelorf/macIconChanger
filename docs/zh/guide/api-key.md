# API 密钥

在线搜索图标需要 [macosicons.com](https://macosicons.com/) 的 API 密钥。没有密钥也可以使用本地图片。

## 获取 API 密钥

1. 访问 [macosicons.com](https://macosicons.com/) 并注册账号。
2. 在账号设置中申请 API 密钥。
3. 复制密钥。

<!-- ![获取 API 密钥](../../public/images/api-key.png) -->

## 输入密钥

1. 打开 IconChanger。
2. 进入 **设置** > **高级**。
3. 在 **API Key** 中粘贴密钥。
4. 点击 **测试连接** 验证是否有效。

<!-- ![API 密钥设置](../../public/images/api-key-settings.png) -->

## 不使用 API 密钥

没有 API 密钥也可以更改图标：

- 使用本地图片（点击 **从本地选择** 或拖放图片）
- 使用应用自带的图标（显示在"本地"区域）

## 故障排除

如果 API 测试失败：
- 检查密钥是否正确（没有多余空格）
- 确认网络连接正常
- macosicons.com API 可能暂时不可用
