# 显示设置

在 **设置** > **显示** 中配置 IconChanger 的视觉外观。

## 外观

在 **系统**、**浅色** 或 **深色** 模式之间选择。系统选项会跟随你的 macOS 外观设置。

### 自定义图标徽章

开启 **显示自定义图标徽章** 后，侧边栏中已应用自定义图标的应用会显示一个绿色对勾标记。

## Dock 预览

Dock 预览会实时展示自定义图标在 macOS Dock 中的效果。

### 预览模式

- **仅在 Dock 筛选时** — 仅在按 Dock 应用筛选时显示 Dock 预览
- **始终** — 始终在详情区域顶部显示 Dock 预览

### 壁纸背景

开启 **显示壁纸背景** 后，Dock 预览会显示你的桌面壁纸作为背景，模拟真实的 Dock 外观。

启用后会出现两个额外的滑块：

- **壁纸扩展** (50–120) — 控制壁纸超出 Dock 区域的扩展范围
- **壁纸模糊** (28–40) — 控制 Dock 背后壁纸的模糊强度

### Dock 玻璃效果强度 (macOS 26+)

在 macOS Tahoe 及更高版本上，**Dock 玻璃效果强度** 滑块 (0–1) 可控制预览中 Liquid Glass Dock 框架的透明度。

## 语言

从 30 种支持的语言中选择。更改在重新启动应用后完全生效。
