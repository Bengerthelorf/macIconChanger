## Points forts

- 🎨 **Icônes personnalisées** — Changez l'icône de n'importe quelle app via GUI ou CLI, avec intégration [macOSicons.com](https://macosicons.com/)
- 🖥️ **Aperçu du Dock** — Prévisualisez en temps réel vos icônes personnalisées dans le Dock
- 📂 **Icônes de dossiers** — Personnalisez aussi les icônes de vos dossiers
- 🔄 **Service en arrière-plan** — Restauration automatique selon un calendrier ou après une mise à jour
- ⌨️ **Outil CLI** — Interface en ligne de commande complète pour le scripting et l'automatisation
- ⭐ **Favoris et historique** — Sauvegardez et retrouvez vos choix d'icônes
- 🌍 **30 langues** — Localisation mondiale

## Installation rapide

### Homebrew

```bash
brew install Bengerthelorf/tap/iconchanger
```

### Manuel

Téléchargez le dernier DMG depuis [Releases](https://github.com/Bengerthelorf/macIconChanger/releases/latest), glissez dans Applications, et lancez.

> **Besoin d'aide ?** Consultez le guide de [démarrage rapide](https://bengerthelorf.github.io/macIconChanger/guide/getting-started), ou parcourez la [documentation complète](https://bengerthelorf.github.io/macIconChanger/).
