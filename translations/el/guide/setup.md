# Αρχική ρύθμιση

Το IconChanger χρειάζεται δικαιώματα διαχειριστή για να αλλάζει εικονίδια εφαρμογών. Κατά την πρώτη εκκίνηση, η εφαρμογή προσφέρει να το ρυθμίσει αυτόματα.

## Αυτόματη ρύθμιση (Συνιστάται)

1. Εκκινήστε το IconChanger.
2. Κάντε κλικ στο κουμπί **Setup** όταν σας ζητηθεί.
3. Εισαγάγετε τον κωδικό διαχειριστή σας.

Η εφαρμογή θα εγκαταστήσει ένα βοηθητικό σενάριο στη διαδρομή `/usr/local/lib/iconchanger/` (ιδιοκτησία `root:wheel`) και θα ρυθμίσει έναν περιορισμένο κανόνα sudoers ώστε να εκτελείται χωρίς να ζητά κωδικό κάθε φορά.

## Ασφάλεια

Το IconChanger χρησιμοποιεί αρκετά μέτρα ασφαλείας για την προστασία του βοηθητικού αγωγού:

- **Βοηθητικός κατάλογος ιδιοκτησίας root** — Τα βοηθητικά αρχεία βρίσκονται στη διαδρομή `/usr/local/lib/iconchanger/` με ιδιοκτησία `root:wheel`, αποτρέποντας μη εξουσιοδοτημένες τροποποιήσεις.
- **Επαλήθευση ακεραιότητας SHA-256** — Το βοηθητικό σενάριο επαληθεύεται έναντι ενός γνωστού hash πριν από κάθε εκτέλεση.
- **Περιορισμένος κανόνας sudoers** — Η καταχώρηση sudoers παρέχει πρόσβαση χωρίς κωδικό μόνο στο συγκεκριμένο βοηθητικό σενάριο, όχι σε αυθαίρετες εντολές.
- **Καταγραφή ελέγχου** — Όλες οι λειτουργίες εικονιδίων καταγράφονται με χρονοσφραγίδες για ιχνηλασιμότητα.

## Χειροκίνητη ρύθμιση

Αν η αυτόματη ρύθμιση αποτύχει, μπορείτε να τη ρυθμίσετε χειροκίνητα:

1. Ανοίξτε το Terminal.
2. Εκτελέστε:

```bash
sudo visudo -f /etc/sudoers.d/iconchanger
```

3. Προσθέστε την ακόλουθη γραμμή:

```
ALL ALL=(ALL) NOPASSWD: /usr/local/lib/iconchanger/helper.sh
```

## Επαλήθευση ρύθμισης

Μετά τη ρύθμιση, η εφαρμογή θα πρέπει να εμφανίζει τη λίστα εφαρμογών στην πλαϊνή μπάρα. Αν εμφανιστεί ξανά η οθόνη ρύθμισης, η διαμόρφωση ενδέχεται να μην εφαρμόστηκε σωστά.

Μπορείτε να επαληθεύσετε τη ρύθμιση από τη γραμμή μενού: κάντε κλικ στο μενού **...** και επιλέξτε **Check Setup Status**.

## Περιορισμοί

Εφαρμογές που προστατεύονται από το System Integrity Protection (SIP) του macOS δεν μπορούν να αλλάξουν εικονίδιο. Αυτός είναι περιορισμός του macOS και δεν μπορεί να παρακαμφθεί.

Συνήθεις εφαρμογές που προστατεύονται από το SIP:
- Finder
- Safari (σε ορισμένες εκδόσεις macOS)
- Άλλες εφαρμογές συστήματος στη διαδρομή `/System/Applications/`
